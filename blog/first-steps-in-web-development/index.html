Probably you have wondered at some point in your life, what is web development? (Or not haha). Here I will give a short introduction to what it is and why it is so important nowadays.

Web development, also known as website development, includes all the tasks associated with developing websites for hosting via the internet. Web development can go from creating a simple static page to a complex web application, it will depend on the software complexity of the code. Let us see this in detail.

Certain web standards technologies help us to build websites, the most used and the ones we are going to learn about in this post are HTML, CSS, and JavaScript. Let us start with the two web design languages for building websites: HTML and CSS.

It forms the blocks that define the layout, format, and critical components of a page. Though HTML is essential to structure information, tasks such as changing the background colour of a section in our page or dimensioning images and icons become difficult and require a lot of work if we are using only HTML. There is when the necessity of a language to add styles to the information already positioned arises.
